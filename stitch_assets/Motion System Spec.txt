# Corpus Motion System Specification

## Background: Evidence Constellation
- **Logic**: Continuous, slow procedural drift using the WebGL shader.
- **Interaction**: Nodes expand slightly (1.2x) and increase glow intensity (1.5x) within a 200px radius of the mouse cursor.

## Agent Trace: Reasoning Flow
- **Activation**: Nodes light up in sequence (intake -> plan -> retrieve, etc.).
- **Transitions**: As a node activates, a 'pulse' (a bright point of light) travels from the previous node to the next along the curved connecting thread over 600ms (ease-in-out).
- **Current State**: The active node pulses between 0.8 and 1.2 opacity with a 2s duration.

## Citations & Interactions
- **Citation Chips**: On hover, chip lifts +4px on Z-axis, glow expands.
- **Citation Cards**: Ease-in (scale 0.95 to 1.0, opacity 0 to 1) from behind the chip.
- **Input Glow**: When query is processing, the send icon and input border pulse with the violet-teal gradient.

## Global Physics
- **Easing**: Custom cubic-bezier(0.2, 0.8, 0.2, 1) for all panel reveals.
- **Inertia**: Elements have a soft-decay settling motion when appearing.
